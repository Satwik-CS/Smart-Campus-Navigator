Import Campus Data

Use this page to import or update campus data through CSV files.

Import Buildings

Upload a CSV file containing building data.

Required CSV format: code,name,description,floors

Example: A,A Block,Main Administrative Block,4

Import Location Types

Upload a CSV file containing location type data.

Required CSV format: name,description

Example: Classroom,Regular classroom for lectures

Import Locations

Upload a CSV file containing location data.

Required CSV format: name,room_number,floor,building_code,type_name,description,capacity,x_coordinate,y_coordinate,facilities

Example: Dean's Office,A-101,1,A,Office,Office of the Dean,5,100,150,"AC, Computer"

Note: Buildings and location types must be imported first!

Import Paths

Upload a CSV file containing path data.

Required CSV format: from_room,to_room,distance

Example: A-101,A-102,10

Note: Locations must be imported first!

Clear All Data

Clear all campus data from the database. This action cannot be undone!
