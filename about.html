About Campus Navigator

An intelligent campus navigation system designed to help students, faculty, and visitors easily find their way around Graphic Era Deemed University.

The Project

The Smart Campus Navigator is a comprehensive web-based application designed to help students, faculty, staff, and visitors efficiently navigate the Graphic Era Deemed University campus. This system combines powerful backend algorithms written in C with a user-friendly web interface to provide detailed information about campus locations, including classrooms, laboratories, offices, and other facilities.

Our Mission

To create an intuitive navigation solution that reduces the time and stress involved in finding locations within the campus, empowering new students and visitors to navigate confidently without assistance.

Key Features

Building-Based Navigation

Click on specific blocks to explore their facilities.

Advanced Search

Find locations by name, room number, or type.

Detailed Information

Comprehensive details about each location.

Interactive Maps

Visual representation of campus buildings.

Technology Stack

The Campus Navigator is built using a combination of modern technologies:

  • Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
  • Backend: Python, Flask, C
  • Database: PostgreSQL with SQLAlchemy ORM
  • Maps: Interactive SVG-based campus visualization

Development Team

This project was developed as part of the curriculum at Graphic Era Deemed University, combining expertise in algorithms, database design, and web development.

The integration of C algorithms with Python web development demonstrates the practical application of computer science fundamentals to solve real-world navigation challenges.

How It Works

  1. Select a building block or use the search function
  2. Filter by floor, location type, or search term
  3. View detailed information about each location
  4. Find connected and nearby locations
  5. Use the interactive map for visual guidance

Administrators can easily import and update campus data through the Import Data section using CSV files.

About Graphic Era Deemed University

Excellence in Education

Graphic Era Deemed University (GEU) is a premier educational institution located in Dehradun, Uttarakhand, India. Established in 1993 as Graphic Era Institute of Technology, it was accorded university status in 2008 and has been recognized as a Deemed University by the UGC.

With a focus on academic excellence, innovation, and holistic development, GEU has established itself as one of the leading universities in Northern India.
